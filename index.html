Hello, I'm Aleksandr

UX/UI Design Student & Self-Taught Game Designer/Developer

My journey started with game development: I spent years taking courses, prototyping mechanics, and designing levels. It taught me to think in systems — how small details shape the user’s experience, whether in a game menu or a mobile app.

Now I’m expanding into UI/UX design, studying how people interact with interfaces beyond games. This website is part of that learning process — I built it without templates to understand how code and design connect.

Interested in projects where logic meets creativity: from gameplay loops to intuitive interfaces. If you’re working on something in that space, let’s talk!
