# Personal Website - HTML/CSS/JS Version (PWP_2_3)

Это обновленная версия персонального веб-сайта с улучшенным дизайном проектов, созданная с использованием только HTML, CSS и JavaScript без фреймворков.

## 📁 Структура файлов

```
PWP_2_3/
├── index.html              # Главная страница
├── project-details.html    # Страница деталей проекта
├── styles.css             # Все стили CSS
├── script.js              # JavaScript функциональность
├── images/                 # Папка с изображениями
├── fonts/                  # Папка со шрифтами
├── temp/                   # Временные файлы
└── README.md              # Это руководство
```

## 🚀 Как запустить

1. **Простой способ**: Откройте `index.html` в любом браузере
2. **Локальный сервер** (рекомендуется для разработки):
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (если установлен live-server)
   npx live-server
   ```

## �🎯 Кастомизация

### 1. **Основная информация**
Отредактируйте `index.html`:
- Замените "Your Name" на ваше имя
- Обновите email адреса
- Измените описания и тексты

### 2. **Цвета и темы**
В `styles.css` найдите CSS переменные:
```css
:root {
  --primary-color: #2563eb;    /* Основной цвет */
  --secondary-color: #64748b;  /* Вторичный цвет */
  --accent-color: #f59e0b;     /* Акцентный цвет */
  /* ... другие цвета */
}

[data-theme="dark"] {
  /* Цвета для темной темы */
}
```

### 3. **Проекты**
В `script.js` измените данные в объекте `projectData`:
```javascript
const projectData = {
  uxui: [
    {
      id: 1,
      title: "Название проекта",
      image: "путь-к-изображению.jpg",
      description: "Описание проекта..."
    }
  ]
}
```

### 4. **О себе**
В `script.js` обновите массив `aboutData`:
```javascript
const aboutData = [
  {
    id: 1,
    title: "Заголовок",
    content: "Ваш текст..."
  }
]
```
